## 実用例

### 基本的なレイアウト

```tsx
// カードコンポーネント
<CoreComponent p="6" bg="white" borderRadius="lg" elevation="2" border="field">
  <h3>カードタイトル</h3>
  <p>カードの内容がここに入ります</p>
</CoreComponent>
```

### Flexboxレイアウト

```tsx
// ナビゲーションバー
<CoreComponent
  as="nav"
  display="flex"
  justifyContent="between"
  alignItems="center"
  p="4"
  bg="primary"
  color="white"
>
  <CoreComponent as="h1" fontSize="20" fontWeight="bold">
    ロゴ
  </CoreComponent>
  <CoreComponent display="flex" gap="4">
    <CoreComponent as="a" href="/">
      ホーム
    </CoreComponent>
    <CoreComponent as="a" href="/about">
      会社概要
    </CoreComponent>
    <CoreComponent as="a" href="/contact">
      お問い合わせ
    </CoreComponent>
  </CoreComponent>
</CoreComponent>
```

### グリッドレイアウト

```tsx
// カードグリッド
<CoreComponent display="grid" gridTemplateColumns="3" gap="6" p="8">
  {items.map((item) => (
    <CoreComponent
      key={item.id}
      p="4"
      bg="white"
      borderRadius="md"
      border="field"
      elevation="1"
    >
      <h4>{item.title}</h4>
      <p>{item.description}</p>
    </CoreComponent>
  ))}
</CoreComponent>
```

### レスポンシブデザイン

## ベストプラクティス

### 1. プロパティの優先順位

```tsx
// ✅ 良い例：一貫性のある順序
<CoreComponent
  // レイアウト
  display="flex"
  flexDirection="col"
  gap="4"
  // スペーシング
  p="6"
  m="4"
  // サイジング
  w="full"
  maxW="800px"
  // タイポグラフィ
  fontSize="16"
  lineHeight="150"
  // カラー・ボーダー
  bg="white"
  border="field"
  borderRadius="md"
  // その他
  elevation="1"
  cursor="pointer"
>
  コンテンツ
</CoreComponent>
```

### 2. レスポンシブ値の活用

```tsx
// ✅ 良い例：段階的なレスポンシブ
<CoreComponent
  fontSize={{ sp: "14", tb: "16", pc: "18" }}
  p={{ sp: "3", tb: "4", pc: "6" }}
  gap={{ sp: "2", tb: "4", pc: "6" }}
>
  レスポンシブコンテンツ
</CoreComponent>

// ❌ 悪い例：一つの値のみ
<CoreComponent fontSize="16" p="4" gap="4">
  レスポンシブでないコンテンツ
</CoreComponent>
```

### 3. トークン値の活用

```tsx
// ✅ 良い例：デザイントークンを使用
<CoreComponent
  p="6"        // 24px
  fontSize="16" // 16px
  gap="4"      // 16px
  borderRadius="md"
  elevation="2"
>
  トークンを使用したコンポーネント
</CoreComponent>

// ❌ 悪い例：カスタム値を多用
<CoreComponent
  p="24px"
  fontSize="16px"
  gap="16px"
  borderRadius="4px"
  boxShadow="0 4px 6px rgba(0, 0, 0, 0.1)"
>
  カスタム値を多用したコンポーネント
</CoreComponent>
```

### 4. セマンティックなHTML要素

```tsx
// ✅ 良い例：適切なHTML要素を使用
<CoreComponent as="article" p="6" bg="white">
  <CoreComponent as="h2" fontSize="24" fontWeight="bold" mb="4">
    記事タイトル
  </CoreComponent>
  <CoreComponent as="p" fontSize="16" lineHeight="160">
    記事の内容
  </CoreComponent>
</CoreComponent>

// ❌ 悪い例：divを多用
<CoreComponent p="6" bg="white">
  <CoreComponent fontSize="24" fontWeight="bold" mb="4">
    記事タイトル
  </CoreComponent>
  <CoreComponent fontSize="16" lineHeight="160">
    記事の内容
  </CoreComponent>
</CoreComponent>
```

### 5. パフォーマンスの最適化

```tsx
// ✅ 良い例：必要なプロパティのみ
<CoreComponent
  display="flex"
  alignItems="center"
  gap="4"
  p="4"
>
  シンプルなコンポーネント
</CoreComponent>

// ❌ 悪い例：不要なプロパティを含む
<CoreComponent
  display="flex"
  alignItems="center"
  justifyContent="start" // デフォルト値なので不要
  gap="4"
  p="4"
  m="0" // デフォルト値なので不要
  w="auto" // デフォルト値なので不要
>
  冗長なコンポーネント
</CoreComponent>
```

### 6. アクセシビリティの考慮

```tsx
// ✅ 良い例：アクセシビリティを考慮
<CoreComponent
  as="button"
  p="4"
  bg="primary"
  color="white"
  borderRadius="md"
  cursor="pointer"
  // アクセシビリティ属性
  aria-label="送信ボタン"
  role="button"
  tabIndex={0}
>
  送信
</CoreComponent>
```

## よくある使用パターン

### ボタン

```tsx
const Button = ({ variant = "primary", size = "md", children, ...props }) => {
  const variants = {
    primary: { bg: "primary", color: "white" },
    secondary: { bg: "secondary", color: "white" },
    outline: { bg: "transparent", color: "primary", border: "primary" },
  };

  const sizes = {
    sm: { p: "2", fontSize: "14" },
    md: { p: "3", fontSize: "16" },
    lg: { p: "4", fontSize: "18" },
  };

  return (
    <CoreComponent
      as="button"
      borderRadius="md"
      cursor="pointer"
      fontWeight="medium"
      {...variants[variant]}
      {...sizes[size]}
      {...props}
    >
      {children}
    </CoreComponent>
  );
};
```

### カード

```tsx
const Card = ({ variant = "default", padding = "6", children, ...props }) => {
  const variants = {
    default: { bg: "white", elevation: "1" },
    elevated: { bg: "white", elevation: "3" },
    outlined: { bg: "transparent", border: "field" },
  };

  return (
    <CoreComponent
      borderRadius="lg"
      p={padding}
      {...variants[variant]}
      {...props}
    >
      {children}
    </CoreComponent>
  );
};
```

### レイアウトコンテナ

```tsx
const Container = ({
  maxWidth = "1200px",
  padding = "6",
  children,
  ...props
}) => (
  <CoreComponent w="full" maxW={maxWidth} mx="auto" px={padding} {...props}>
    {children}
  </CoreComponent>
);
```
